# CLash Of Clans REPLICA

## Player Controls
### Movement Controls
```
W-Move Up<br>
A-Move Left<br>
S-Move Down<br>
D-Move Right<br>
```
### How to attack?
```
Space-Attack
R-Activate Rage Spell<br>
H-Activate Heal Spell<br>
```
### Troop calls
```
1,2,3 for barbarians 
4,5,6 for archers 
7,8,9 for ballons
```
### To exit the game
```
Q-Quit
```

## To run
python3 main.py

## For Replay
python3 replay.py
