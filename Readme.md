# CLash Of Clans REPLICA
## How to run?
```
python3 main.py
```
## For Replay
```
python3 replay.py
```
## Player Controls
### Movement Controls
```
W-Move Up
A-Move Left
S-Move Down
D-Move Right
```
### How to attack?
```
Space-Attack
R-Activate Rage Spell
H-Activate Heal Spell
```
### Troop calls
```
1,2,3 for barbarians 
4,5,6 for archers 
7,8,9 for ballons
```
### To exit the game
```
Q-Quit
```
